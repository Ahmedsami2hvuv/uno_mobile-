# سيرفر API لتطبيق أونو (مشروع مستقل)

هذا المشروع **مستقل تماماً**. لا يعدّل ولا يعتمد على مستودع بوت أونو. ملفات البوت تبقى كما هي.

## ماذا فيه؟

- **api_server.py** — واجهة تسجيل ودخول وبروفايل للتطبيق.
- **database.py** — اتصال بقاعدة البيانات عبر `DATABASE_URL`.
- **requirements.txt** — المتطلبات.
- **Procfile** — لأمر التشغيل على Railway.

## كيف أنشئ مشروعاً جديداً على Railway؟

### 1) إنشاء مستودع جديد على GitHub

1. ادخل إلى GitHub واختر **New repository**.
2. اسم المستودع مثلاً: **uno-app-api**.
3. لا تضع أي ملفات (Empty repo).
4. انسخ الملفات من مجلد **uno-app-api** هذا إلى المستودع الجديد:
   - `api_server.py`
   - `database.py`
   - `requirements.txt`
   - `Procfile`
   - `README.md`
5. اعمل Commit و Push.

### 2) إنشاء مشروع جديد على Railway

1. ادخل إلى **railway.com** → من الصفحة الرئيسية اضغط **"+ New"**.
2. اختر **"Deploy from GitHub repo"**.
3. اختر المستودع **uno-app-api** (المستودع الجديد، وليس uno).
4. بعد إنشاء المشروع، اضغط على الخدمة (Service) التي ظهرت.
5. من القائمة: **Variables** (أو Settings → Variables):
   - اضغط **Add Variable** أو **Raw Editor**.
   - أضف متغيراً واحداً اسمه **DATABASE_URL** وقيمته **نفس رابط قاعدة البيانات** من مشروع البوت:
     - من مشروع **uno** في Railway افتح خدمة **قاعدة البيانات** (Postgres) وانسخ **DATABASE_URL** أو رابط الاتصال.
     - الصق نفس القيمة في مشروع **uno-app-api**.
6. من **Settings** للخدمة: **Generate Domain** أو **Public Networking** → انسخ الرابط (مثل `https://uno-app-api-production-xxxx.up.railway.app`).

### 3) ربط التطبيق

في مشروع **uno-mobile** على جهازك افتح **src/api.js** وضع في **BASE_URL** الرابط الذي نسخته من Railway لمشروع uno-app-api.

---

## ملخص

| المشروع | المحتوى | هل نعدّل البوت؟ |
|---------|---------|------------------|
| **uno** (البوت) | البوت + قاعدة البيانات | لا — لا نلمس ملفاته |
| **uno-app-api** (جديد) | سيرفر API فقط | لا — مشروع منفصل |
| **uno-mobile** (التطبيق) | تطبيق الموبايل | نضع فيه رابط مشروع uno-app-api |

بذلك تطبيق الأونو يتصل بمشروع جديد (uno-app-api)، والمشروع الجديد يتصل **بنفس قاعدة البيانات** اللي يستخدمها البوت، والحساب يكون واحداً.
