# تطبيق أونو للموبايل

تطبيق **هاتف** (أندرويد / آيفون) مبني على فكرة بوت أونو من [مستودع uno](https://github.com/Ahmedsami2hvuv/uno). نصوص القوائم والقوانين والحاسبة مأخوذة من البوت (i18n.py) ليكون فتح التطبيق كفتح البوت نفسه.

## التشغيل على الهاتف

1. **تثبيت المتطلبات (مرة واحدة)**
   - [Node.js](https://nodejs.org) (إصدار LTS)
   - تطبيق **Expo Go** على هاتفك من [Google Play](https://play.google.com/store/apps/details?id=host.exp.exponent) أو [App Store](https://apps.apple.com/app/expo-go/id982107779)

2. **تشغيل المشروع**
   ```bash
   cd C:\Users\lenovo\uno-mobile
   npm install
   npx expo start
   ```

3. **فتح التطبيق على الهاتف**
   - عند ظهور QR Code في الطرفية:
     - **أندرويد:** افتح تطبيق Expo Go وامسح الـ QR.
     - **آيفون:** افتح الكاميرا وامسح الـ QR ثم اختر "فتح في Expo Go".
   - أو على نفس الشبكة اختر "Run on Android device" / "Run on iOS simulator" إن كان المحاكي أو الجهاز متصل.

## المحتوى

- **اللعب ضد البوت (AI)** — نفس قواعد البوت (أونو، جوكر +4 وتحدي، منع، تدوير، +2، اختيار لون).
- **القوانين** — شرح مختصر للعبة.
- **حاسبة النقاط** — لأوراق أونو الحقيقية (أرقام، أكشن، جوكر).

## البناء لتوزيع التطبيق (APK / AAB أو iOS)

لإنشاء ملف تثبيت يمكن نشره على المتاجر أو توزيعه:

- **Expo EAS Build:** سجّل حساباً على [expo.dev](https://expo.dev) ثم:
  ```bash
  npx eas build --platform android
  ```
  لبناء نسخة أندرويد (APK أو AAB حسب الإعداد).

- للآيفون تحتاج حساب Apple Developer وإعدادات إضافية في EAS.

## دعم العربية

التطبيق يستخدم **RTL** (من اليمين لليسار) وواجهة بالعربية.

## الفكرة الكاملة

الفكرة المستخرجة من البوت (غرف أصدقاء، لعب عشوائي، بروفايلات، شارات) موثقة في مجلد المشروع السابق `uno-app` في ملف **CONCEPT.md** — يمكن الرجوع إليه لتوسيع التطبيق لاحقاً.
